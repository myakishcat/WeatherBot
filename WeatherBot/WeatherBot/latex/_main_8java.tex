\doxysubsection{Detailed Description}
Главный класс для запуска Telegram бота. 

Этот класс отвечает за инициализацию и регистрацию бота в Telegram API. Он запускает процесс работы с Telegram API, создавая экземпляр бота и регистрируя его с использованием {\ttfamily Telegram\+Bots\+Api}.
