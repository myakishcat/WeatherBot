\doxysubsection{Detailed Description}
Класс для обработки ответа от HTTPS сервера. 

Этот класс обрабатывает ответ от сервера, получая код ответа и данные, если код успешен. Он позволяет проверить код ответа и извлечь данные из потока, если код ответа равен 200 (OK).
