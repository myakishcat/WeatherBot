Detailed Description

Класс клиента для установления соединения с сервером по HTTP.

Класс Client представляет собой клиент для работы с HTTP-соединением. Он позволяет устанавливать соединение с указанным URL, настраивать методы и заголовки запросов, а также получать ответы от сервера.

Этот класс отправляет HTTP-запрос к API погоды, получает ответ, а затем парсит данные для дальнейшего использования.
