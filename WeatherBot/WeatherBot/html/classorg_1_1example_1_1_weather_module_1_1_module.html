Detailed Description

Модуль для запроса и обработки данных погоды.

Этот класс отправляет HTTP-запрос к API погоды, получает ответ, а затем парсит данные для дальнейшего использования.

Выполняет запрос к API погоды и возвращает объект WeatherData.

Этот метод формирует запрос к внешнему API погоды, отправляет его, получает ответ и парсит данные о погоде в объект WeatherData.
